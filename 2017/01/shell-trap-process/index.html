Shell执行中信号捕获的处理方式

介绍

当shell在执行过程中，使用CTRL+C可以中断当前shell的执行，强制中断执行时可能需要做些清理任务，此时可以捕获中断信号进行shell进程结束前的处理。

脚本

#!/bin/bash

# 定义中断执行方法
function exitmode() {
	echo -e "clear temp file"
	exit 1
}

# 注册信号处理
trap exitmode SIGINT SIGHUP

# 模拟一个10秒的任务
sleep 10

信号

在unix系统中使用命令kill -l查看所有支持的信号
