    Pyinstaller打包python应用

    2017-10-11
    • 打包参数
    • 使用方法

    打包参数

    参数说明
    -w直接打包会带调试窗口，使用-w参数可以进行屏蔽
    -C使用调试界面，与-w参数相反，缺省
    -F打包成独立的exe文件
    -D与-F相反，生成多个以来文件，缺省
    -p搜索模块路径
    -i改变生产的exe图标

    使用方法

    pyinstaller -w main.py
pyinstaller -w -F main.py
pyinstaller -w -F -p [绝对路径] main.py
pyinstaller -w -F -i main.ico -p [绝对路径] main.py
