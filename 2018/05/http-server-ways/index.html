    快速创建简易HTTP服务器的方式

    2018-05-07
    • 1. Python 方式
    • 2. Php 方式
    • 3. Ruby 方式
    • 4. Npm 方式

    1. Python 方式

    python2 -m SimpleHTTPServer 8000
python3 -m http.server 8000

    2. Php 方式

    php -S 0.0.0.0:80000

    3. Ruby 方式

    ruby -rwebrick -e \ 
"WEBrick::HTTPServer.new(:Port => 8000, :DocumentRoot => Dir.pwd).start"

    4. Npm 方式

    可以全局安装npm install http-server -g ，也可以使用npm install方式安装

    "scripts": {
	"start": "http-server -c-1 -p 8081"
},
"devDependencies": {
	"http-server": "^0.11.1"
}
