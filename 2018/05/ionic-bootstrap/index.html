    在 ionic 中集成 bootstrap 框架

    2018-05-13
    • 安装bootstrap依赖
    • 配置复制任务
    • 创建任务文件
    • 导入bootstrap样式
    • 使用bootstrap组件

    Bootstrap 是流行的前端 UI 框架，具备丰富的组件库及色彩搭配。当你觉得 ionic 原生组件不满足于要求时，可以尝试下使用 Bootstrap 来进行页面构建。

    本文以认为你已经准备好了一个标准的 ionic 工程了。

    安装bootstrap依赖

    进入 ionic 项目根路径，执行以下 npm 命令进行 boostrap 的安装

    $ npm install bootstrap --save

    安装好之后检查下 package.json 文件，bootstrap 是否已经安装好以及其版本是否正确。

    ...
  "dependencies": {
    ...
    "bootstrap": "^4.1.1",
  },
...

    配置复制任务

    编辑根路径下的 package.json 文件，在文件末尾增加 config 任务，此任务用于在构建时拷贝相关样式。

    "config": {
  "ionic_sass" : "./sass.config.js"
}

    创建任务文件

    在项目根路径下创建 sass.config.js 文件，内容如下：

    module.exports = {
  includePaths : [
    'node_modules/ionic-angular/themes',
    'node_modules/ionic-angular/fonts',
    'node_modules/ionicons/dist/scss',
    'node_modules/bootstrap/scss'
  ]
};

    导入bootstrap样式

    在 src/theme/variables.scss 文件中使用 @import 命令导入 bootstrap 样式表

    ...
@import "bootstrap";
...

    使用bootstrap组件

    上述操作完成后，就可以在 bootstrap components 中选择需要的组件直接在 html 页面中使用即可。

    <ion-content padding>

  <div class="container">
    <div class="row">
      <button type="button" class="btn btn-primary btn-lg btn-block">Block level button</button>
      <button type="button" class="btn btn-secondary btn-lg btn-block">Block level button</button>
    </div>
  </div>
    
</ion-content>
