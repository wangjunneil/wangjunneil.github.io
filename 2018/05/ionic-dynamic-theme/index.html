    让 ionic 支持动态主题切换

    是否曾经在某些 APP 上看见如 白天 与 夜间 模式，且选择不同色彩搭配。本篇文章将在 ionic 上实现类似效果。为了简单性，这里只实现两种效果。

    创建主题样式文件

    在 src/theme 目录中创建两个样式文件，theme.dark.scss 和 theme.light.scss，这两个文件定义全局的样式表。

    [theme.dark.scss]

    .dark-theme {
    .header .toolbar-title {
        color: #fff;
    }

    .header .toolbar-background {
        border-color: #fff;
        background-color: #000;
    }

    ion-content {
        background-color: #000;
        color: #fff;
    }
}

    [theme.light.scss]

    .light-theme {
    .header .toolbar-title {
        color: #000;
    }

    .header .toolbar-background {
        border-color: #000;
        background-color: #fff;
    }

    ion-content {
        background-color: #fff;
        color: #000;
    }
}

    从上面两个样式表可以看出，我们定义了全局的 header 以及 content 体的主题色调，为了便于演示，这里使用黑白搭配，用户可以根据实际情况自由进行颜色处理。

    导入主题样式文件

    在 src/theme/variables.scss 中使用 @import 命令将两个样式文件进行导入

    @import "theme.light";
@import "theme.dark";

    创建切换主题服务

    新建一个 settings 类的服务，用于切换主题风格，这里在根路径下执行命令创建：

    $ ionic g provider settings

    上面命令执行完成后，会生成 src/providers/settings/settings.ts 的服务文件，增加切换主题的方法，最后如下：

    import { Injectable } from '@angular/core';
import { BehaviorSubject } from 'rxjs/Rx';

@Injectable()
export class SettingsProvider {
  private theme: BehaviorSubject<String>;  
  constructor() {
    // 默认为白天主题
    this.theme = new BehaviorSubject('light-theme');
  }
  setActiveTheme(val) { // 设置主题
    this.theme.next(val);
  }
  getActiveTheme() {  // 获取主题
    return this.theme.asObservable();
  }
}

    设置启动默认主题

    编辑 src/app/app.component.ts 文件，使用创建的 settings 服务设置启动时缺省的主题

    // ...
import { SettingsProvider } from '../providers/settings/settings';

@Component({
  templateUrl: 'app.html'
})
export class MyApp {
  selectedTheme: String; // 当前主题变量
    
  constructor(platform: Platform, 
    statusBar: StatusBar, 
    splashScreen: SplashScreen,
    private settings: SettingsProvider) {
      // 设置启动时的缺省主题
      this.settings.getActiveTheme().subscribe(val => this.selectedTheme = val);
      // ...
  }
}

    完成上述配置后，接下来在 src/app/app.html 中使用变量 selectedTheme 设置全局的主题样式

    <ion-nav [root]="rootPage" [class]="selectedTheme"></ion-nav>

    实现切换主题事件

    在页面中使用两个按钮来激活切换主题的事件，为了测试，这里就在 home.html 中增加文本和按钮用于展示

    <ion-content padding>
  <p text-center>what's fuck man ?</p>

  <button ion-button full icon-left (click)="switchTheme('light-theme')">
    <ion-icon name="cloud-outline"></ion-icon>
    白天模式
  </button>
  <button ion-button full icon-left (click)="switchTheme('dark-theme')">
    <ion-icon name="cloud"></ion-icon>
    夜间模式
  </button>
</ion-content>

    编辑好页面控件后，下面实现在 home.ts 文件中实现 switchTheme() 方法

    import { SettingsProvider } from '../../providers/settings/settings';

// ...
export class HomePage {
  constructor(public navCtrl: NavController, private settings: SettingsProvider) {
  }
  // 按钮切换主题的方法
  switchTheme(theme: String) {
    this.settings.setActiveTheme(theme);
  }
}
