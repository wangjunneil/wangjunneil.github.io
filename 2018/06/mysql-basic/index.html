    1. 配置yum源地址

    centos 中有的版本可以直接运行 yum install mysql mysql-server 的方式进行安装，但安装的是 MariaDB，MariaDB是mysql的一个分支，这里为了确保安装的是mysql版本，所以配置源地址。

    # 下载源地址文件
$ wget http://dev.mysql.com/get/mysql57-community-release-el7-8.noarch.rpm
# 安装mysql的yum源
$ yum localinstall mysql57-community-release-el7-8.noarch.rpm
# 检查mysql源是否安装成功
$ yum repolist enabled | grep "mysql.*-community.*"

    上面的源默认是mysql5.7的版本，若希望安装其他版本，如5.6的版本，可以编辑 vi /etc/yum.repo/mysql-community.repo 的文件，将指定版本的 enabled 更改即可，0 是禁用， 1 是启用，如下：

    2. 执行mysql的安装

    $ yum install -y mysql-community-server

    安装路径与配置文件

    名称值
    配置文件/etc/my.cnf
    日志文件/var/log/mysqld.log
    启动脚本/usr/lib/systemd/system/mysqld.service
    socket文件/var/lib/mysql/mysql.sock
    pid文件/var/run/mysqld/mysqld.pid

    3. 启动与检测mysql

    $ systemctl start mysqld
$ systemctl status mysqld

    4. 设置开机自启动

    $ systemctl enable mysqld
$ systemctl daemon-reload

    5. 修改root密码

    mysql 启动后会在 /var/log/mysqld.log 文件中生成一个默认的root密码，可以使用命令 grep 'temporary password' /var/log/mysqld.log 找到

    使用默认的root密码登录并修改

    $ mysql -u root -p

# 修改root密码的两种方式
mysql> ALTER USER 'root'@'localhost' IDENTIFIED BY '1q@W3e$R';
mysql> set password for 'root'@'localhost' = password('MyNewPass4!');

    如果密码设置过于简单会触发mysql的密码安全检查插件(validate_password)，这是5.7默认安装的，如下：

    可以在mysql环境中使用 show variables like '%password%'; 查看当前的密码策略，这些可以在 my.cnf 中进行修改

    6. 添加远程用户

    默认只是允许 root 账号在本地 localhost 上登录，不建议 root 可以远程登录，为了安全起见，应该添加授权一个远程用户

    mysql> GRANT ALL PRIVILEGES ON *.* TO 'vinny'@'%' IDENTIFIED BY '1q@w3e$R' WITH GRANT OPTION;
mysql> flush privileges;

    7. 修改默认编码

    编辑 vi /etc/my.cnf 配置文件，添加编码配置，如下：

    [mysqld]
character_set_server=utf8
init_connect='SET NAMES utf8'
